NAME
fmt(3f) - [M_msg] convert any intrinsic to a string using specified format (LICENSE:PD)

SYNOPSIS
function fmt(value,format) result(string) 

    class(*),intent(in),optional :: value
    character(len=*),intent(in)  :: format
    character(len=:),allocatable :: string

DESCRIPTION
fmt(3f) converts any standard intrinsic value to a string using the specified format.

OPTIONS
value value to print the value of. May be of type INTEGER, LOGICAL, REAL, DOUBLEPRECISION, COMPLEX, or CHARACTER.
format format to use to print value. It is up to the user to use an appropriate format. The format does not require being surrounded by parenthesis.

RETURNS
string A string value

EXAMPLES
Sample program: 

    program demo_fmt
    use :: M_msg, only : fmt
    implicit none
    character(len=:),allocatable :: output


       output=fmt(10,"’[’,i0,’]’")
       write(*,*)’result is ’,output


       output=fmt(10.0/3.0,"’[’,g0.5,’]’")
       write(*,*)’result is ’,output


       output=fmt(.true.,"’The final answer is [’,g0,’]’")
       write(*,*)’result is ’,output
end program demo_fmt

Results: 

    result is [10]
    result is [3.3333]
    result is The final answer is [T]
