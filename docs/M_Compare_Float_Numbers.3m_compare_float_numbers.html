PURPOSE
Module containing routines to perform equality and relational comparisons on floating point numbers.

CATEGORY
Utility

LANGUAGE
Fortran-95

CALLING SEQUENCE
USE M_Compare_Float_Numbers

CONTAINS
.EqualTo.
  Relational operator to test the equality of floating point numbers.
.GreaterThan.
  Relational operator to test if one operand is greater than another.
.LessThan.
  Relational operator to test if one operand is less than another.
Compare_Float:
  Function to compare floating point scalars and arrays with adjustable precision tolerance.
