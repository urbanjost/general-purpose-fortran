NAME
poippf(3f) - [M_datapac:PERCENT_POINT] compute the Poisson percent point function

SYNOPSIS
SUBROUTINE POIPPF(P,Alamba,Ppf)

DESCRIPTION
POIPPF(3f) computes the percent point function value at the precision precision value P for the Poisson distribution with REAL tail length parameter = alamba.

the poisson distribution used herein has mean = alamba and standard deviation = sqrt(alamba). this distribution is defined for all discrete non-negative integer x--x = 0, 1, 2, ... .

this distribution has the probability function 

       f(x) = exp(-alamba) * alamba**x / x!.
the poisson distribution is the distribution of the number of events in the interval (0,alamba) when the waiting time between events is exponentially distributed with mean = 1 and standard deviation = 1.

note that the percent point function of a distribution is identically the same as the inverse cumulative distribution function of the distribution.
