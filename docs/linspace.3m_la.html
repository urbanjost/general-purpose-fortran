NAME
linspace(3f) - [M_LA] - return a vector of linearly spaced values

CONTENTS

Synopsis
Description
Options
Returns
Example

SYNOPSIS
y = linspace(x1,x2) y = linspace(x1,x2,n)

DESCRIPTION
linspace returns a vector of linearly spaced values from x1 to x2 inclusive. It gives direct control over the number of points and always includes the endpoints, the results being the same as [(x1+i*(x2-x1)/(n-1),i=0,n-1)] if n>1 and [x1,x2] if n<=1.

OPTIONS
X1,X2 X1 and X2 are the upper and lower bound of the values returned. The options can be of type REAL or INTEGER, but must be of the same type.

RETURNS
LINSPACE
  The returned row vector starts with X1 and ends with X2, returning N evenly spaced values.

EXAMPLE
Sample program: 

   program demo_linspace
   use M_LA,  only : linspace
   implicit none
   character(len=*), parameter :: gen=’(*(g0, 1x))’
      write( *, gen ) linspace(  0,      9,    10 )
      write( *, gen ) linspace( 10.0,   20.0,  11 )
      write( *, gen ) linspace( 11.1d0, 12.1d0, 5 )
      write( *, gen ) linspace( 11.1,   12.1,   5 )
   end program demo_linspace
